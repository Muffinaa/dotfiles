### Keybinds

> Notice: I still have to bind some stuff

|Key|Action  |
|--|--|
| **SPC w** | Save document |
| **SPC q** | Quit document |
| **SPC s** | Reload config |
| **SPC ff** | Telescope search files |
| **SPC fb** | Telescope search buffers |
| **SPC fg** | Telescope live grep |
| **SPC c** | Toggle file explorer |
| **SPC v** | Focus File explorer |
